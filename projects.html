Lifedrinker

Lifedrinker was a research project created during a semester long project class at Iowa State University. The aim of this project was to identify a chosen object of interest at range using a camera feed, a Convolutional Neural Network (CNN), and a tracking algorithm to assist in the detection, identification, and tracking of the target.

This solution was then implemented as a rail-mounted tactical scope attachment for a Nerf Blaster. The attachment provides the user with real-time feedback on their aim using a side mounted screen.

Clark

Clark is a 50 pound semi-autonomous robot running on ROS.

Clark uses a variety of sensors to determine its position, including an Xbox 360 Kinect, two cameras, a 9-axis IMU, and various other feedback sources. Clark is driven by a 6-wheel West Coast style drop center drive, giving it superb traction on most surfaces and the ability to turn in place. Built using 80/20 T-slot aluminum extrusion, Clark features a very flexible, but structurally rigid design, placing an emphasis on modularity and upgradability.

7Words

7Words is a web tool intended to mass lyric check a set of songs. It uses the Genius API and the Spotify API to search through a playlist of songs for foul language that could go against FCC broadcasting guidelines. This application was originally created for Iowa State's student run radio station, 88.5 FM KURE Ames Alternative, but has been open sourced for use by any interested party.

Gazpacho

Gazpacho is an Android app intended to help find, document, and share recipes.

Focusing on collaboration, Gazpacho allows users to document and share any changes or substitutions they have made to recipes featured in the app. Built using Android Studio, Gazpacho places an emphasis on the social aspect of cooking, and brings the world's cooking skills to your kitchen.

Endeavor, aka Smokey 11

Endeavor featured a two-speed pneumatic gearbox on its 6-CIM motor drivetrain. This allowed Endeavor to accelerate quickly, while also having a incredible amount of torque and pushing power. It also had a large rotating arm with a pneumatically driven second joint to collect game pieces.

Smokey 9.5

Smokey 9.5 was a traditional beetleweight tank-bot created during my sophomore year of high school.

It was controlled by both a Raspberry Pi and an Arduino with input provided wirelessly using a video game controller. Smokey 9.5 also has a gripper on the front to allow it to manipulate objects around it. It also can run autonomously using distance sensors to navigate its surroundings.

ORTrepORT

OrtReport is an Android application that is dedicated to tracking an individual's food waste over time, while also providing information on how to live sustainably.

Created for Iowa State University's Fall 2018 Hackathon, OrtReport was developed as an introductory project to Kotlin and Android Studio, as our group had never worked with either before.
